# Vulnerability Assessment Practice

This repository documents my hands-on practice in conducting vulnerability assessments, focusing on methodologies and tools used in environments like TryHackMe.

## 🔍 Objectives
- Practice identifying and analyzing common vulnerabilities
- Develop remediation strategies
- Create standardized documentation for assessments
- Build a professional workflow and portfolio

## 🛠 Tools
- Nmap
- Nessus (when applicable)
- OpenVAS
- Nikto
- Burp Suite (for web-focused scans)

## 🗂️ Structure
Each folder under `tryhackme-scans/` represents a room or practice environment. It contains:
- Target Info
- Raw scan output
- Vulnerability analysis
- Suggested remediations
- Screenshots
